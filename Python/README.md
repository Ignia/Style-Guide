### Capitalization
| Type                          | Case                          | Example
| ------------------------------|-------------------------------|-------------------------------
| Class                         | `PascalCase`                  | `ShoppingCart`
| Private Properties            | `_camelCase`                  | `_shoppingCart`
| Local Variables               | `camelCase`                   | `shoppingCart`
| Parameters                    | `camelCase`                   | `checkoutStatus`
| Constants                     | `UPPER_SNAKE_CASE`            | `IMAGES_DIRECTORY`
